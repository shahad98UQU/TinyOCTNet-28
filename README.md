# TinyOCTNet-28: Deep Learning for Medical Diagnostics

## Project Description
Designed, coded, and trained a customized lightweight Convolutional Neural Network (CNN) for automated retinal disease classification using the OCTMNIST dataset.

## Key Features
* Optimized the architecture (including SE attention and CoordConv layers) for edge deployment, achieving high diagnostic accuracy while reducing computational overhead and latency.
* Implemented multi-class classification for 4 diagnosis categories: CNV, DME, DRUSEN, and NORMAL[cite: 5].

## Dataset
* Based on the **OCTMNIST** dataset (MedMNIST v2), which contains optical coherence tomography (OCT) images for retinal diseases[cite: 5].
